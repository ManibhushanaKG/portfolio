## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/ManibhushanaKG/portfolio.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
